Bot Information

This launcher will apply all MongoDB truthiness and string formatting fixes before starting the bot.

The bot requires the following environment variables:

  • MONGODB_URI: MongoDB connection string
  • DISCORD_TOKEN: Discord Bot token

These can be set in the Replit Secrets tab.

Command Line Usage

# Start the bot in production mode
./launch.sh

# Apply fixes only
./launch.sh fix-only

# Check environment variables
./launch.sh check-env
